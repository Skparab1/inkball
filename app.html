Ball

Direct the ball into the target of the same color

Target

Direct the ball of the same color into this target

Release point

The balls will be released from this point

Obstable Block

The balls will deflect off these obstacle blocks

Release timer

Indicates when a ball will be released

Timed blocks

Obstacles that appear and disappear after set intervals

Booster

Accelerates the ball in the indicated direction

Boost Strip

Long accelerates the ball in the indicated direction

Lock block

Appears and disappears when a same colored ball is scored

Break block

Block that can be broken by a ball of the same color.
